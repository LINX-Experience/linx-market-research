## Scoring Frameworks

All entities are scored 0-100 on standardized rubrics:

- **Operators:** ICP Fit Score (asset proximity, revenue, referral dependency, commission arbitrage, brand, tech, approachability)
- **Competitors:** Threat Score (funding, feature parity, market overlap, momentum, velocity)
- **HOA Communities:** Travel Affinity Score (home value, marina access, aviation, culture, concierge, residents, migration, proximity)
