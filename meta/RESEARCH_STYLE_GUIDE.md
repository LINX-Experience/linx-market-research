# LINX Research Engine — Style Guide v1.0

> Applies to ALL research output: operator profiles, market pulse, battle cards, HOA profiles, digests.

---

## Universal Rules

### 1. Source Everything
- Every factual claim needs a source (URL, publication name, or report title)
- If no source: write "Source not confirmed" or "Based on prior knowledge — verify"
- NEVER invent a URL. NEVER fabricate a statistic.
- If WebSearch returns no results on a topic: "No recent data found via web search."

### 2. 5W1H on Every Signal
Every market signal, competitive move, or industry development must answer:
- **WHO** — the specific entity (company name, person name, fund name)
- **WHAT** — exactly what happened (specific, not vague)
- **WHEN** — date, timeframe, or effective date
- **WHERE** — geography, market segment, vertical
- **WHY** — the underlying driver (not just "they announced it" but the strategic reason)
- **HOW** — the mechanism by which this affects LINX / Stein's ventures
- If any element is unknown: write "Unknown" — never guess

### 3. Confidence Levels
- **CONFIRMED:** Multiple sources corroborate. URL provided. Fact-checked.
- **LIKELY:** Single credible source or strong inference from confirmed data.
- **UNVERIFIED:** Single source, unconfirmed, or based on prior knowledge. Flag explicitly.

### 4. LINX Lens + Cross-Venture Lens
Every finding must be interpreted through:
- What does this mean for LINX? (GTM, product, positioning, fundraise)
- What does this mean for AXIOM? (superyacht economy, charter market, investment thesis)
- What does this mean for Stein Studios? (production industry, expedition media, brand content)
- What does this mean for Stein personally? (wealth building, network, reputation, expedition world)

Not every signal touches all four. But consider all four before writing.

### 5. Density Over Volume
- Every sentence must be actionable or contextual. No filler.
- "The luxury travel market continues to grow" = filler. Cut it.
- "$544B in leisure travel from top 10% US households, up 8.1% YoY (Resonance 2026)" = density. Keep it.

### 6. Anti-Sycophancy in Research
- Never present data optimistically to make LINX look good.
- If a competitor is winning, say so. If a market is shrinking, say so.
- "This validates LINX's thesis" is only allowed if it genuinely does — with evidence.
- Challenge the LINX thesis in every weekly synthesis. What would DISPROVE it?

## Format Standards by File Type

### Operator Profiles
- ICP Fit Score (0-100) with all 7 dimensions in a table
- Archetype assignment with reasoning
- Top 3 objections with scripted responses
- Intro path (specific person or channel)
- Est. GMV and LINX revenue (mark confidence level)
- Every claim sourced

### Battle Cards
- Threat Score (0-100) with 5-dimension breakdown
- "Recent Moves" section: date-stamped, newest first
- "LINX Counter-Strategy" must be specific, not generic
- Rewritten to current state weekly (not infinite append)

### Market Pulse Daily
- 5W1H framework on every signal
- Categorized: TREND / EVENT / THREAT / OPPORTUNITY
- Urgency scored: CRITICAL / HIGH / MEDIUM / LOW
- Top 3 actions at the end (specific, not "monitor")

### HOA Community Profiles
- Travel Affinity Score (0-100) with 8 dimensions
- Primary vertical affinity assignment
- Estimated addressable LINX users (with methodology)
- Acquisition strategy (specific channel, not generic)

### Executive Digests
- Follow DIGEST_STANDARD.md exactly
- 5-8 minute read (1,500-2,500 words)
- Cross-venture coverage mandatory
- Bottleneck Check mandatory
